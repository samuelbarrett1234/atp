30 // try to match the trial expression to the pattern expression
31 // this will be done, if possible, by replacing free variables
32 // in 'pattern' to match 'trial'.
33 // this function may be useful for determining if one can apply
34 // a particular theorem to a statement.
35 // precondition: pattern and trial are both not SyntaxNodeType::EQ.
36 // postcondition: returns an assignment of free variable IDs (in
37 // pattern) to syntax tree expressions which would make 'pattern',
38 // after the subsitution, equal to 'trial' up to the order of the
39 // free variables (we don't care about swapping ID's of the free
40 // variables.)
