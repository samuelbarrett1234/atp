Detailed Description

Contains the Boost Spirit and Boost Qi grammar specifications for the equational logic parsing.

Author
Samuel Barrett

This file contains the parser for the equational logic statements, using Boost Spirit for constructing the parser from a CFG. This can then produce a parse tree, which can then be turned into a syntax tree.

Note
Boost's Qi library doesn't seem to let us export the parsers, so they aren't being exported here.
Using this prints out parsing info to std::cout to help debug parser errors: #define BOOST_SPIRIT_DEBUG
