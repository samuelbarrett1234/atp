Detailed Description

Interfaces for search heuristic functions.

Author
Samuel Barrett

Uninformed search strategies won't get very far, thus we use a heuristic function (or several) to help guide the search. Heuristic functions are represented as objects because they might need persistent information, like model parameters or even just the knowledge kernel. Their operations are also vectorised.
