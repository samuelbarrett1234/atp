Detailed Description

Query builders are objects which create SQL statements for specific operations common to ATPDatabase users, and ensure the SQL syntax is correct for the particular SQL version and database implementation being used.

Author
Samuel Barrett

The main necessity of these objects is to (i) reduce the amount of code needed for query string writing in the IDatabase implementations, and (ii) have a uniform interface for creating these queries regardless of the difference in SQL semantics across DBMSs.
