Detailed Description

Interface for proof solving strategies.

Author
Samuel Barrett

Solvers are the core interface for the search procedure. They are given a fixed language and knowledge kernel, and then you can set target statements for them, which they will attempt to prove by using the reductions available from the knowledge kernel. Solvers are vectorised for efficiency; each thread is thus trying to prove many statements at once. Furthermore, solvers are designed to work in 'steps', allowing you to execute arbitrary code in-between.
