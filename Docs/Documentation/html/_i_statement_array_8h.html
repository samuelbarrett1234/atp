Provides the IStatmentArray interface for collections of statement objects.

\detailed This file contains an interface to a class representing a one-dimensional array of statements. This interface is mostly useful for efficiency purposes - arrays of IStatement objects can be slow. It can also implement lazy semantics where possible.
