Detailed Description

Implementation of the IKnowledgeKernel for equational logic.

Author
Samuel Barrett

\detailed The Knowledge Kernel object contains the functions which tell you what is/isn't allowed in equational logic, and functions for logical inference etc. Currently, proofs in equational logic operate only in "iff" steps, so proofs can be read in both directions. This is for simplicity but can be a bit limiting. Furthermore, we don't support propositional operators (yet) so there is no support for and/or/not. Finally, there is no support for showing that two expressions are NOT equal; in other words, we cannot prove a statement to be false, only find a proof if it is true.
