Detailed Description

This class is for converting arrays of statements into matrices of sequences of integer observation values.

Since efficiency is very important for training, it is worth the design cost to handle each kind of logic separately and to get an efficient conversion to an observation sequence.

Note
Symbol IDs are represented in the range 0..N-1, and free variables are represented by N, where N=symb_ids.size() is the size of the array of symbol IDs given.
