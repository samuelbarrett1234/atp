Detailed Description

This process transfers data from ProofSetupEssentials to ProofEssentials, and loads "helper theorems" from the database as well.

To summarise, it: creates the knowledge kernel, creates the solver, loads the helper theorems from the database, and then hands over the ProofEssentials data.
