Detailed Description

The core object of this library - basically represents a collection of tables.

The main purpose of this object is to help constructing IDBOperation objects, which allow you to query database state and update the database. Databases are intrinsically tied to a kind of logic, and all of their tables use only that logic type. (In other words, each database holds its own Language object).
