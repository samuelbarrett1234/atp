Detailed Description

Interface for building a query to select theorems from the database.

See also
RANDOM_THM_SELECTION

This query is only ready to build when the limit and context have been set. By default, it only returns theorems with proofs, however you can set it so that it only returns unproven theorems, instead, if you wish.
