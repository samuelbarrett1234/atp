Detailed Description

This class represents a set of free variable IDs.

Why not just use std::set<size_t>? Because this class takes advantage of the fact that free variable IDs are usually close together (dense), so we subtract the minimum ID from all of them, and then store it as a bitmap.

Note
Don't need to export this class as it is fully inlined.
