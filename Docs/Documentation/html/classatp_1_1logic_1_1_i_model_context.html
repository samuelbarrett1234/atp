Detailed Description

A model context is a collection of definitions and axioms.

Model contexts are loaded from a file, and become immutable after that. They are created via the ILanguage interface. They produce knowledge kernels, which are related to model contexts.

Note
Every symbol has a name, an ID, and an arity. The ID is pretty much just a hash of the name, and is only there for efficiency reasons (so that Statement objects don't need to store strings for names).
