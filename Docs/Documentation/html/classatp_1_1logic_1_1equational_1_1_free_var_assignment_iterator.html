Detailed Description

Produces statement successors by iterating over possible assignments of free variables not covered by an existing mapping.

An object for iterating over all possible assignments of the remaining free variables that weren't covered by the partial mapping resulting from a match. This iterator works by picking a remaining free variable ID arbitrarily, and enumerating values for it, but also recursing (creating child FreeVarAssignmentIterator objects) to enumerate the remaining free variables, if there are any. If there are not any, then this implementation is trivial, and only has one item to iterate over.
