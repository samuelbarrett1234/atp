Check if the conclusions follow in one logical step from the corresponding premises.

Precondition
p_premise->size() == p_concl.size() and valid(p_premise) and valid(p_concl)
Postcondition
For each (premise, conclusion) pair, returns true if that conclusion follows from that premise in exactly one step (equivalently, if the conclusion is a successor of the premise.)

size_t atp::logic::equational::KnowledgeKernel::get_integrity_code ( ) const
overridevirtual

Characterises the information provided to this knowledge kernel in a single integer code.

If two knowledge kernels agree on their code, then they agree on the return results of all the functions below. (This code is computed similarly to a hash code of all the user definitions, axioms, and available theorems.)

◆ valid()

bool atp::logic::equational::KnowledgeKernel::valid ( StatementArrayPtr  p_stmts) const
overridevirtual

Type-check the given statements.

Postcondition
for each statement, check whether it is valid in terms of the logic, axioms and definitions in this knowledge kernel. Statements are only allowed to be used if they are valid. Return true iff ALL statements in the array are valid. Note that a necessary condition for validity is that they must be using the same language as that which created this knowledge kernel.
