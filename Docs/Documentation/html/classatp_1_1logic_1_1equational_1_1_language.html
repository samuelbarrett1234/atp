StatementArrayPtr atp::logic::equational::Language::normalise ( const StatementArrayPtr &  p_stmts)
overridevirtual

Reduce an array of statements to some "normal form".

This, more specifically, means: reducing free variables, forcing an ordering about the equals sign, and reducing to equivalence classes under the statement equivalence relation.

Returns
A new array of statements with the above effects applied (n.b. the returned array might not be the same length!)
