Detailed Description

The NoRepeatIterator blocks states that have already been seen on the path from the root to the current node. It does so by wrapping around another given iterator.

Note
It doesn't necessarily block states that have been seen before on a different branch - only ones that have occurred from the root to the current proof state node.
