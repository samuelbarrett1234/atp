Detailed Description

For iterating over all subtrees (subexpressions) of a given statement in pre-order traversal, starting with the left-hand expression and ending with the right-hand one.

A statement iterator is just an iterator for the LHS and an iterator for the RHS, which then uses the LHS iterator until it becomes invalid, and when that happens it subsequently just uses the RHS iterator until that becomes invalid too.
