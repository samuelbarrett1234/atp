Detailed Description

This class computes the edit distance of equational logic statements.

This class is optimised in two ways: (i) it is specialised to the logic type so is more efficient, and (ii) caches information between calls.

Note
The returned value is not a distance per se, as it is signed. It is more of a heuristic inspired by edit distance than any true distance metric.
Warning
This class is NOT thread safe, please use a different version of this heuristic for each solver object.

Compute the pairwise edit distance, but looping through the "sub-expressions" of the statements of the first array.

Use this over edit_distance if you are more interested in how far stmt2 is from any sub-part of stmt1, rather than just distances as a whole.

Remarks
This version of the function is not symmetric in its arguments.
Returns
An array A such that A[i][j][k] == edit_distance( kth sub-expression of stmtarr1[i], stmtarr2[j])
