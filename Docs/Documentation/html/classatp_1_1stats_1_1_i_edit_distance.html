Detailed Description

An object for computing edit distance for statements in a particular logic type, which is optimised.

This class is thread-safe.

Compute the pairwise edit distance, but looping through the "sub-expressions" of the statements of the first array.

Use this over edit_distance if you are more interested in how far stmt2 is from any sub-part of stmt1, rather than just distances as a whole.

Remarks
This version of the function is not symmetric in its arguments.
Returns
An array A such that A[i][j][k] == edit_distance( kth sub-expression of stmtarr1[i], stmtarr2[j])

Implemented in atp::stats::EquationalEditDistanceTracker.
