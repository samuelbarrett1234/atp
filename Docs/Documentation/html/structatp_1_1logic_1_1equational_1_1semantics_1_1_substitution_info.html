Detailed Description

Contains data used in some functions below, bundled together for efficiency reasons.

\detailed The immediate_applications and substitute_tree functions below require a lot of data to run, so to save having lots of arguments, and to save recomputing some information several times, we bundle it up in this struct which is shared between the function calls.
